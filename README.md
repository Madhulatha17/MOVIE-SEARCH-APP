# MOVIE-SEARCH-APP

## Description
A React application that searches movies using the OMDb API and allows users to save favorite movies.

## Features
- Search movies by title
- Fetch movie details from OMDb API
- Display poster, year, genre, and IMDb rating
- Save favorite movies
- Favorites stored using Local Storage
- Responsive UI

## Technologies
- React
- CSS
- OMDb API

## Run Project

npm install

npm start
